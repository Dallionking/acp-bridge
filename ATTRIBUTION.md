# Attribution

## Agent Client Protocol (ACP)

This package implements the Agent Client Protocol developed and maintained by Zed Industries.

- Protocol: [Agent Client Protocol](https://agentclientprotocol.com) (Apache-2.0)
- Python SDK: `agent-client-protocol` on PyPI

## Source

Adapted from an open-source ACP CLI client (MIT-licensed). Adapted with namespace changes and identity-neutral messaging conventions for orchestration use cases. Protocol logic, session management, and flow runner patterns are derived from the upstream implementation.
